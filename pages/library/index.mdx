## Intruduction

Pada halaman ini akan menampilkan library yang rata - rata digunakan oleh programmer, baik itu untuk membuat aplikasi, website, dan lain sebagainya. Library yang akan ditampilkan disini banyak menggunakan bahasa pemrograman JavaScript, tetapi tidak menutup kemungkinan akan ada library yang menggunakan bahasa pemrograman lainnya.

Library ini semua bernilai `objektif` dari penulis, jadi jika ada library yang tidak ditampilkan disini kalian dapat mencarinya di google atau menggunakan AI seperti ChatGPT.
Jika kalian ingin menambahkan library yang tidak ada disini, kalian dapat mengirimkan pull request ke repository ini atau gunakan link pada **table of content** di sebelah kanan yang bertuliskan `Question? Give us feedback`.
